Tail Tetranacci

Algorithm TailTetr(k)
Input: Nonnegative integer k
Output: the kth Tetranacci number Fk

FUNCTION Tetranacci(n):
    RETURN TetranacciHelper(n, 0, 0, 0, 1)

FUNCTION TetranacciHelper(n, a, b, c, d):
    IF n == 0:
        RETURN a
    IF n == 1:
        RETURN b
    IF n == 2:
        RETURN c
    IF n == 3:
        RETURN d
    RETURN TetranacciHelper(n - 1, b, c, d, a + b + c + d)
